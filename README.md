Checkers Game in C++ with SFML

Description
This is a simple implementation of the classic Checkers game in C++ using the SFML library. It features a graphical user interface (GUI) where two players can play against each other by controlling their pieces and capturing the opponent's pieces.

Features
- 8x8 checkers board with alternating light and dark squares.
- Player 1 and Player 2 pieces represented by different colors.
- Piece movement and capture logic with highlights for valid moves.
- Win condition when one player captures all of the opponent's pieces.
- Restart option after a game is finished.

Controls
- Left-click to select a piece and move it to a valid square or capture an opponent's piece.
- The game alternates between Player 1 (Red) and Player 2 (Blue) after each move.

Install SFML:
Make sure you have the SFML library installed on your system.
